SE Extended

SE Extended is a fork from the SnapEnhance app that uses the Xposed Framework to enhance your Snapchat experience.

Please note that this project is currently in development, So bugs and crashes may occur. If you encounter any issues, we encourage you to report them. To do this simply visit our issues page and create an issue, make sure to follow the guidelines.
